# curriculum

1. 게임 서버 틀 만들기
  - 첫 시간에는 같이 모여서 멘토가 알려주는 방식으로 로그인 기능을 가진 게임서버를 만든다.
  - 네트워크 프로그래밍은 멘토가 제공하는 네트워크 라이브러리 코드를 사용한다.
  - 패킷에 대한 설명
  - 테스트용 클라이언트에 대한 설명
  - 게임서버 큰 틀이 만들어지고, 앞으로 클라이언트와 주고 받는 기능은 어떻게 만들면 되는지 알 수 있다.
2. 방 입장, 방 나가기, 방 채팅 만들기

3. 오목 게임 컨텐츠 만들기
  - 게임 시작 요청, 게임 시작
  - 오목 두기(타임아웃 적용)
  - 오목 체크, 게임종료
